# ReciprocalLaplacianSquareFactor

!syntax description /TensorComputes/Solve/ReciprocalLaplacianSquareFactor

Fills a tensor with

\begin{equation}
\mathbf{k}^4\cdot f
\end{equation}

where $\mathbf{k}$ is the k-vector. This is the Fourier transform of the squared Laplacian operator.
