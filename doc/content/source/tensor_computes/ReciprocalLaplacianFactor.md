# ReciprocalLaplacianFactor

!syntax description /TensorComputes/Solve/ReciprocalLaplacianFactor

Fills a tensor with

\begin{equation}
-\mathbf{k}^2\cdot f
\end{equation}

where $\mathbf{k}$ is the k-vector. This is the Fourier transform of the Laplacian operator.
